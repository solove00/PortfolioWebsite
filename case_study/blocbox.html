ABOUT BLOCBOX

Cloud service is becoming more and more important nowadays. In the past, we stored information on removable disks and personal computers, but now it is a trend to use cloud services to store files. It is safer, offers more capacity and is more accessible. Blocbox is a cloud storage and file-sync tool. It lets us store all file types including notes, images, and links.

Even though there are many cloud service apps in the marketplace, many of the current apps are lacking in a good user experience. My research found that users prefer a simple design with less features. Too many features sometimes is a problem.I conducted user research, user stories and flows to better understand users’ needs and create a file saving app that has an intuitive user experience. In order to provide the best user experience, I conducted UX research to find out what features were most valuable, and what layout users prefered.


THE RESEARCH

User Research

I designed a user research survey to know my target users. My questionnaire focused on:

  • When you find a link or an image that interests you online, do you save it?
  • How do you save that content
  • What specific tools or sites do you use to bookmark
  • What do you like the most about those tools
  • What do you like least about those tools
  • How often do you write notes to remember things for later
  • Why do you write and keep notes
  • What applications do you use to write and keep notes
  • Do you use Facebook or Twitter account to register or sign up for an account on other websites
